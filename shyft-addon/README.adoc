== Homeassistant addon

This is the shyft home-assistant addon. :-)


=== Installing

* Install plugin.
* Goto github project page - https://github.com/ehmkah/ha-shyft-addon
* There is a link add add-on to home assistant
* !Attention! - its not possible to install the add-on to a homeassistant which is running from the shell (e.g. because you developed your own integration)
